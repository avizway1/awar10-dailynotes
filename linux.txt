/ 		: Root directory : To level directory in linux os, Everything starts here.. 

/root 	: The home directory for root user. (c:\Users\Administratpr)

/home	: Home directory of regular/non-root users. 

/bin	: Binary Executables : Contains all essential binary files used by all users.

/boot	: Boot loader files : LInux kernel and boot config

/dev	: Device files : Contains file the represent hardware devices i.e; hard disks.. 

/etc	: Configuration files : Stores system wide config files (i.e; /etc/passwd, /etc/groups)

/var	: Variable files : Contains data that changes frequently (logs, database, documenttoot path)

/usr	: user programs : contains user level programs and libs.

/sbin	: system binaries : System admin commands (ifconfig, iptables) - Only root/superuser can execute

/lib & lin64	: similar to windows .dll files : Essential shared library files.. 
/proc	: process inforamtion (cpu / RAM / processes)
/opt	: optional software
/mnt	: mount points
/media	: removable media

vim : VI Improved editor : 

Text dirors: vi, vim and nano


Command Mode : This mode is for navigation, deletion and other file operations.
Insert Mode : Used for entering text into the file. 

To switch modes:

command mode --> Insert mode --> press "i"
Insert mode --> command mode --> Press "esc"

To save and quit: 
:wq

Ignore the changes and quit: 
:q!

o		: Open a new line below the current line and enters the insert mode.

j		: Move cursor one line down

:set number		: Sets the line number
: set nonumber	: removes the line number
