
RDS : Relational Database Service : data stores structured manner in tables using rows and columns. 
--> Tabular format
--> CRUD : Create, Read, Update and Delete

NoSQL : Non-Relational Databases : Data stores unstructured or semi structured data in flexible formats i.e; document, key-value, graphs, JSON.



On-Premise Hosted DB : Physical hardware.? OS Installation.? OS Patching.? DB Install.? DB Configuration.? DB backup.? DB HA.? DB FT.?

EC2 Instance Hosted DB : OS AWS Provides.. OS Patching.? DB Install.? DB Configuration.? DB backup.? DB HA.? DB FT.?

RDS : OS AWS, OS Patching AWS, DB Configuration AWS (PS),  DB backup AWS (SS), DB HA AWS (M AZ), DB FT AWS (RR)..
